Sistema Solar: Scretch

O projeto apresentado foi desenvolvido na plataforma Scratch e tem como tema principal o Sistema Solar. Trata-se de uma simulação visual e interativa que representa os planetas orbitando ao redor do Sol, de forma animada e dinâmica.

A construção do projeto foi realizada com blocos de programação visual, permitindo o controle dos movimentos dos planetas em torno do Sol. Para isso, foram utilizados comandos como "girar", "mover" e "sempre repetir", criando uma rotação contínua que simula as órbitas planetárias. Cada planeta foi configurado individualmente, com imagens personalizadas e scripts que definem seu comportamento específico na simulação.

O objetivo principal do projeto é demonstrar, de maneira simples e educativa, o funcionamento do Sistema Solar, facilitando a compreensão dos conceitos de rotação e translação, especialmente para estudantes do ensino fundamental e médio.

O Scratch foi escolhido como ferramenta por sua interface acessível e seu potencial para desenvolver projetos interativos voltados à educação. A proposta une tecnologia e conhecimento científico em uma linguagem de fácil assimilação. O projeto está disponível online na plataforma do Scratch, e pode ser visualizado e remixado por outros usuários, incentivando a troca de conhecimentos e o aprendizado colaborativo.

Minha Biblioteca: Uma Webpage Personalizada

"O Espinheiro Alvar" é um projeto interativo criado com p5.js, uma biblioteca de JavaScript que facilita a criação de arte e animações digitais. O projeto consiste em bolinhas que se multiplicam ao longo do tempo, criando uma dinâmica visual que remete a um crescimento exponencial e caótico, como se fossem partes de uma planta ou organismo se espalhando. A ideia central do projeto é explorar a noção de objetos (neste caso, as bolinhas) se replicando e se espalhando por toda a tela, criando uma sensação de expansão e movimento contínuo.

Em termos de implementação, o código é relativamente simples, mas eficaz para criar o efeito desejado. Na função setup(), a tela é configurada, e uma bolinha inicial é criada no centro. Cada bolinha é representada por um objeto que tem propriedades como posição (x, y), tamanho e um comportamento de atualização, que pode incluir movimentos aleatórios ou outros efeitos. As bolinhas se movem suavemente pela tela, e, a cada ciclo de animação (em p5.js, isso ocorre no draw()), há uma chance de novas bolinhas se "gerarem", ou seja, se multiplicarem, a partir da posição de o

A duplicação das bolinhas é controlada por uma condição aleatória, onde, a cada frame, existe uma probabilidade de que uma bolinha se reproduza em uma nova posição. Isso cria o efeito visual de multiplicação exponencial: à medida que o número de bolinhas cresce, o espaço na tela se preenche rapidamente, fazendo com que o comportamento se torne mais complexo e denso. Esse crescimento pode ser ajustado modificando a probabilidade de multiplicação ou a velocidade com que as bolinhas se movem. Em resumo, "O Espinheiro Alvar" é um projeto que utiliza as possibilidades da programação para criar uma representação visual de multiplicação e expansão, com um foco na interação e na dinâmica entre os elementos da tela. Ele não só demonstra como a programação pode gerar efeitos artísticos e imprevisíveis, mas também permite que o espectador ou usuário vivencie a ideia de crescimento orgânico, como se as bolinhas fossem organismos vivos que se espalham pelo ambiente.
